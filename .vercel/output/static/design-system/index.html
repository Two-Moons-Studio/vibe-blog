Architecture Overview

Page Structure

Three PageHeader variants with consistent typography patterns.

  • Hero - Homepage + Zero to Ship (640px max-width, line-height: 1)
  • Standard - Posts, About, Subscribe, Archive, Tags
  • Minimal - Lighter spacing for utility pages

Container System

Three container patterns for different content types.

  • Wide (1200px) - Homepage only
  • Medium (900px) - Zero to Ship learning page
  • Standard (720px) - All other pages for reading

Editorial Spacing

Generous top spacing creates luxurious editorial feel.

  • Desktop - calc(var(--space-10) + var(--space-8)) = 192px from page top
  • Mobile - calc(var(--space-8) + var(--space-6)) = 96px from page top
  • Consistent - All variants use identical top spacing formula
